How to Run Purple Notes

Important: Purple Notes requires a local web server to run properly due to browser security restrictions on ES modules.

Method 1: Using VS Code Live Server (Recommended)

1. Open VS Code
2. Install the "Live Server" extension:
  • Click the Extensions icon in the sidebar
  • Search for "Live Server"
  • Install "Live Server" by Ritwick Dey
3. Open Purple Notes:
  • Right-click on index.html
  • Select "Open with Live Server"
  • The app will automatically open in your browser

Method 2: Using Terminal Commands

Using Python:
python -m http.server 8000
Using Node.js:
npx live-server
Then open your browser to:
http://localhost:8000
For more information and troubleshooting, please refer to the README.md file in the project directory.
