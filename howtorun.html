How to Run Purple Notes

Recommended Method: Double-click the start.bat / Purple Notes Web Shortcut file to run the application automatically.

Features:
  • Automatic dependency installation
  • Choice between app mode and default browser
  • Multiple browser support in app mode (Brave, Chrome, Edge, Opera GX)
  • Automatic cleanup when closed

Alternative Method

Using VS Code Live Server

1. Install the "Live Server" extension:
  • Click the Extensions icon in the sidebar
  • Search for "Live Server"
  • Install "Live Server" by Ritwick Dey
2. Open Purple Notes:
  • Right-click on index.html
  • Select "Open with Live Server"
Note: The application requires a local server to run properly due to browser security restrictions on ES modules.

For more information and troubleshooting, please refer to docs/architecture.md and docs/update.md in the project directory.
